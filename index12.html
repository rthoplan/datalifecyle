Key Characteristic: Fixed Rows & Columns

This data is organised in a fixed format, much like a traditional spreadsheet. It's rigid, predictable, and easy to query using standard tools.

Example: Student Information System

StudentID FirstName LastName Major
101 Aisha Khan Computer Science
102 Ben Carter Biology
103 Carla Diaz History
Typical File Types: CSV, XLSX

Key Characteristic: Labeled Fields

This data isn't confined to a rigid table, but it contains tags or keys that identify individual pieces of data. This makes it flexible and self-describing.

Example: API Payload (JSON)

Human-readable format common for web APIs and configuration files. Uses key-value pairs.

{
  "orderID": "A9S8D7F6",
  "customer": { ... },
  "items": [ ... ],
  "dispatched": true
}

Example: Document Markup (XML)

A markup language with a tree-like structure. More verbose than JSON but highly extensible.

<order id="A9S8D7F6">
  <customer>
    <name>John Doe</name>
    <email>john.doe@email.com</email>
  </customer>
  <items>
    <item productID="P123" quantity="1"/>
    <item productID="K456" quantity="1"/>
  </items>
</order>

Concept: Columnar Storage (Parquet)

A binary file format optimized for analytics. It stores data in columns, not rows, making queries on specific columns extremely fast because you don't have to read the whole row.

// This is a conceptual representation. The actual file is binary.
--- Column: productID ---
"P123"
"K456"

--- Column: name ---
"Wireless Mouse"
"Mechanical Keyboard"

--- Column: quantity ---
1
1

Key Characteristic: No Predefined Schema

This data has no predefined model or organisation. It's free-form and can be text, media, or other binary data that doesn't fit into a neat structure.

Example 1: Scanned Document (TXT)

Meeting Notes - August 6, 2025

Great discussion today about the Q4 marketing campaign.
Key takeaways:
- Finalize budget by Friday.
- Sarah to mock up new ad creatives.
- Focus on social media engagement.

Example 2: Social Media Image (JPEG)

An image file is a collection of pixels without an explicit, organised schema.

Typical File Types: JPEG, MP4, TXT, PDF
