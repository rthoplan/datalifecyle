The Data Lifecycle

Understanding the journey of data from its creation to its eventual retirement.

1. Generation

The creation of new data from various sources like sensors, user inputs, transactions, or external data streams.

2. Collection

Gathering the raw data from its sources and transferring it to a centralized location for processing and storage.

3. Processing

Cleaning, transforming, and structuring the raw data. This includes handling missing values, normalization, and enrichment.

4. Storage

Persisting the processed data in a secure and accessible database, data warehouse, or data lake for future use.

5. Management

Overseeing the data assets, including governance, security, access control, and ensuring data quality over time.

6. Analysis

Examining data to discover patterns, derive insights, and answer business questions using statistical methods and algorithms.

7. Visualisation & Interpretation

Presenting findings in a clear, graphical format (charts, dashboards) to facilitate understanding and decision-making.

8. Archival/Destruction

Retiring data that is no longer needed. This involves either long-term archiving or secure deletion to comply with policies.
